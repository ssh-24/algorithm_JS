## 시간 복잡도
입력 크기의 값에 대해 단위 연산을 몇 번 수행하는지 계산하여, 알고리즘의 수행시간을 평가하는 방법
- O (빅오) : **최악**의 상황을 고려하여 성능 측정 결과 표현
- Θ (세타) : 평균
- Ω (오메가) : 최선
